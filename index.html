Installation

Prerequisites

  • Modded OFOX
  • ADB & Fastboot
  • A Brain (Very important)

Steps

  1. Flash modded OFOX recovery
  2. Run the partitioning script
  3. Check if Android still starts
  4. Boot OFOX recovery
  5. Execute the msc script
  6. Use Diskpart to assign drive letters
  7. Install Windows
  8. Install Drivers
  9. Create Windows bootloader files
  10. Enable test signing
  11. Disable recovery
  12. Disable integrity checks
  13. Reboot to Android for dual boot setup

FAQ

What devices are supported?

This project currently supports the Redmi Note 9 Pro / 9 Pro India / 10 Lite / 9S / 9 Pro max India / Poco M2 Pro.

Is this project stable?

This project is currently in the beta stage, which means that while most of the hardware works, some components may not be fully functional yet.

What are the risks involved?

Modifying your device and its partition table can be risky and may result in a bricked device. Please follow the instructions carefully and proceed at your own risk.

Can I dual boot Android and Windows?

Yes, it is possible to set up a dual boot system with Android and Windows on your device.
